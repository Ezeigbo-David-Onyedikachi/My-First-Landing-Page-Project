# Aptech Learning Center - Interactive Landing Page

A professional, high-performance landing page built to showcase educational pathways at Aptech. 
This version includes custom JavaScript to enhance user experience and interactivity.

## Live Demo
https://ezeigbo-david-onyedikachi.github.io/My-First-Landing-Page-Project/

## Project Overview
This project serves as a functional prototype for an educational platform. It demonstrates 
a mastery of responsive layouts and DOM manipulation to create a seamless browsing experience.

## Interactive Features (Powered by JavaScript)
* **Dynamic Navigation:** A custom-built mobile menu toggle that handles responsive states.
* **Scroll-to-Reveal Animations:** Sections fade into view as the user scrolls, creating a modern feel.
* **Form Validation:** Basic client-side checks for the "Contact/Inquiry" section.
* **Smooth Scrolling:** Enhanced navigation between page sections for a premium user experience.
